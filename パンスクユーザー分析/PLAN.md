# PLAN.md — パンスクユーザー分析プロジェクト

## プロジェクト概要
パンスク定期便の登録×配送データを分析し、事業改善につなげるためのレポートを作成する。

## 分析したいこと（ユーザーの声）
1. **長く続けているユーザーはどんなユーザーか？**
   - 年代・性別・エリア・プランの傾向を知りたい
2. **コホート毎のLTVはどうなのか？**
   - 登録月別のLTV・継続率・離脱率を把握したい
3. **広告経由で登録したユーザーのLTVはどうなのか？**
   - 広告 vs オーガニックのLTV比較
4. **広告費が全体的にどのような影響を与えているか？**
   - 高予算期・低予算期・停止期で登録数・LTV・継続率がどう変わるか
