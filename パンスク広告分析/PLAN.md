# PLAN.md — パンスク Meta広告 クリエイティブ分析

## プロジェクト概要

パンスクのMeta広告（Facebook/Instagram）におけるクリエイティブ（CR）の実績を多角的に分析し、今後の広告運用改善に向けた示唆を導く。

## データ概要

- **対象期間**: 2026/01/01 ~ 2026/03/15（74.5日間）
- **データソース**: パンスクCR実績_2026_03_16.xlsx
- **対象キャンペーン**: 3種類（ASC、ASC_検証、INT）
- **クリエイティブ数**: 97本
- **総消化金額**: 約447,860円

## 分析の目的

1. **効果的なクリエイティブの特定** — どのCRがCVを獲得できているか
2. **クリエイティブタイプ別パフォーマンス** — 画像/動画/カルーセルの比較
3. **キャンペーン別パフォーマンス** — ASC vs INT の差異
4. **コスト効率の評価** — CPA・CTR・CPMによる多面評価
5. **改善提案の導出** — 停止すべきCR・強化すべきCRの特定

## 分析の進め方

### Phase 1: データ整理・集計（完了 2026/03/17）
- ローデータの把握
- 成果の良い・悪いCRシートの確認
- キャンペーン別・CR種別の基本集計

### Phase 2: 多角的分析（完了 2026/03/17）
- CPA/CTR/CPM/CVRによる総合評価
- クリエイティブタイプ別分析
- 効率性マトリクス作成・分析レポートExcel作成

### Phase 3: データ基盤構築 × LTV分析（完了 2026/03/19）
- BigQuery / GA4 / Firestoreの接続状況調査
- gaClientId × user_pseudo_id JOIN の実証
- Meta広告経由ユーザーの継続率・LTV算出
- 全流入源コホート分析（2025-03〜現在）
- Meta ASC ROI検証（12ヶ月で2.1倍黒字を確認）

### Phase 4: CR単位LTV追跡（次のステップ）
- 代理店に utm_content={{ad.name}} の設定を依頼
- 設定後1ヶ月以降にCR単位のコホート分析を実施

## 主要な問い

- 最も費用対効果の高いCRはどれか？
- 動画・画像・カルーセルではどの形式が優れているか？
- INTキャンペーンは有効に機能しているか？
- 予算を集中すべきCRはどれか？
- 停止を検討すべきCRはどれか？
